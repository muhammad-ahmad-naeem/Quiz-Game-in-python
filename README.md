# 🧠 Quiz Time

A simple desktop trivia quiz built with **Python** and **Tkinter**, styled with a warm beige theme.

![Python](https://img.shields.io/badge/Python-3.8%2B-blue)
![Tkinter](https://img.shields.io/badge/GUI-Tkinter-8C6A3B)

## Features

- Clickable option cards with hover effects
- Instant feedback — correct answers turn green, wrong ones turn red
- Progress bar to track your position in the quiz
- Results screen with a full answer review
- Restart button to play again

## Getting Started

```bash
git clone https://github.com/<your-username>/quiz-time.git
cd quiz-time
python quiz_app.py
```

No extra installs needed — just Python's built-in `tkinter`.
> Linux users may need to install it first: `sudo apt install python3-tk`

## Customizing Questions

Edit the `QUESTIONS`, `OPTIONS`, and `ANSWERS` lists at the top of `quiz_app.py` to create your own quiz.
